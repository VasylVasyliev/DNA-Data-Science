PROJECT: GENOMIC K-MER ANALYSIS
------------------------------
Description:
A Python-based tool for analyzing DNA sequences using k-mer searching.

Key Features:
- Fuzzy search (Hamming distance = 1 mismatch allowed).
- GC-content filtering (range 40-60%).
- Automated visual mapping of markers.
- Cross-file uniqueness validation.

Results:
- orf.txt: Found 43 unique markers (Best: AGGTCTT).
- dna.txt: Found 67 unique markers (Best: GCTTCTG).
